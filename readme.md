ZJsBridge
===
> 一套完整的native-bridge-web协议与实现，清晰规范的开发Hybrid App

[ZJsBridge项目](https://github.com/hcanyz/ZJsBridge)

## ZJsBridge能做什么
- 对web端提供完整的js-sdk，形成sdk版本概念
- 对native提供jsapi组件化实现能力
- 交互过程数据完整性校验

## 什么场景下需要使用ZJsBridge
- 较多的web与native交互，需要统一native对外api
- native组件化，需要不能模块提供不同native api
